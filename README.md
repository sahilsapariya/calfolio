# Getting started with Calfolio backend


### Install dependencies

#### Create DB in MySQL

1. login to mysql
2. create a database named `calfolio`

```
CREATE DATABASE calfolio;
```

### Clone the repo

```bash
git clone https://github.com/sahilsapariya/calfolio.git

cd calfolio

mvn clean install

mvn spring-boot:run
```
